# rmbr - remember it

A modern note-taking and project management app built with Next.js.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

- Next.js 16 (App Router)
- BlockNote (Notion-style editor)
- Zustand (State management)
- Tailwind CSS
- dnd-kit (Drag & Drop)
- Lucide React (Icons)

## Features

- Hierarchical page navigation
- Block-based rich text editor
- Kanban board with drag & drop
- Document/Board view toggle
- Local persistence
